# متجر إلكتروني احترافي

موقع إلكتروني احترافي للتسوق مشابه لأمازون مع نظام إدارة المنتجات والمصادقة.

## المميزات

- ✅ واجهة مستخدم احترافية وعصرية
- ✅ نظام تسجيل دخول وإنشاء حساب يعمل 100%
- ✅ إدارة المنتجات (إضافة، تعديل، حذف)
- ✅ رفع صور المنتجات
- ✅ الوضع الليلي/النهاري (Dark/Light Theme)
- ✅ البحث في المنتجات
- ✅ تصميم متجاوب (Responsive)
- ✅ قاعدة بيانات SQLite
- ✅ API كامل مع Express.js

## المتطلبات

- Node.js (الإصدار 14 أو أحدث)
- npm

## التثبيت

1. تثبيت الحزم المطلوبة:
```bash
npm install
```

2. تشغيل الخادم:
```bash
npm start
```

3. فتح المتصفح على:
```
http://localhost:3000
```

## استخدام الموقع

### تسجيل الدخول
1. اضغط على زر "تسجيل الدخول"
2. أدخل البريد الإلكتروني وكلمة المرور
3. أو أنشئ حساب جديد من خلال تبويب "إنشاء حساب"

### إضافة منتج
1. سجل الدخول أولاً
2. اضغط على زر "+ إضافة منتج"
3. املأ بيانات المنتج (الاسم، السعر، الكمية، الوصف)
4. ارفع صورة المنتج (اختياري)
5. اضغط "حفظ المنتج"

### تعديل/حذف منتج
- بعد تسجيل الدخول، ستظهر أزرار "تعديل" و "حذف" لكل منتج

### تغيير الثيم
- اضغط على أيقونة 🌙/☀️ في الأعلى للتبديل بين الوضع الليلي والنهاري

## API Endpoints

### المصادقة
- `POST /api/register` - إنشاء حساب جديد
- `POST /api/login` - تسجيل الدخول
- `GET /api/user` - الحصول على بيانات المستخدم

### المنتجات
- `GET /api/products` - الحصول على جميع المنتجات
- `GET /api/products/:id` - الحصول على منتج محدد
- `POST /api/products` - إضافة منتج جديد (يتطلب تسجيل دخول)
- `PUT /api/products/:id` - تعديل منتج (يتطلب تسجيل دخول)
- `DELETE /api/products/:id` - حذف منتج (يتطلب تسجيل دخول)

## التقنيات المستخدمة

- **Backend**: Node.js, Express.js
- **Database**: SQLite3
- **Authentication**: JWT (JSON Web Tokens), bcryptjs
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **File Upload**: Multer

## الأمان

- كلمات المرور مشفرة باستخدام bcrypt
- JWT tokens للمصادقة
- تحقق من صحة المدخلات
- حماية من رفع الملفات الضارة

## التطوير المستقبلي

- تقييمات المنتجات

- لوحة تحكم للمشرفين
